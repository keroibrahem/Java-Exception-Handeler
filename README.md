# Java Exception Handling Exercises

## Overview
This repository contains exercises for practicing **Java Exception Handling**, including:
- Basic try-catch blocks
- Handling arrays and files
- Custom exceptions
- Exception propagation and `throws` keyword
- Finally block and nested try-catch

These exercises are structured in **three levels**:

### Level 1: Basic Try-Catch
1. Divide two numbers and handle `ArithmeticException`.
2. Convert a string to integer and handle `NumberFormatException`.
3. Handle `NullPointerException` when calling a method on `null`.

### Level 2: Arrays and Files
4. Access an array element and handle `ArrayIndexOutOfBoundsException`.
5. Read a file and handle `FileNotFoundException`.
6. Handle multiple exceptions using multiple catch blocks.

### Level 3: Custom Exceptions
7. Create a custom `InvalidAgeException` for age validation.
8. Propagate exceptions between methods.
9. Use the `throws` keyword for methods throwing `IOException`.

### Bonus / Advanced
10. Demonstrate `finally` block execution.
11. Nested try-catch example.
